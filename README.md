# Antigravity WSL Setup

This repository provides the setup script for running Google Antigravity on Windows Subsystem for Linux (WSL).

## Purpose

The `setup-antigravity-wsl.sh` script automates the installation and configuration of Google Antigravity in a WSL environment.

## Detailed Setup Guide

For complete step-by-step instructions on setting up Antigravity on WSL, check out my blog post:

**[Google Antigravity WSL Setup Guide](https://smaxiso.web.app/blog/google-antigravity-wsl-guide)**

## Usage

```bash
chmod +x setup-antigravity-wsl.sh
./setup-antigravity-wsl.sh
```

## About

This script streamlines the process of getting Google Antigravity up and running in your WSL environment, handling all necessary dependencies and configurations.
